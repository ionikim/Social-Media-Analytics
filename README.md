Project Idea: Temporal Network Evolution in Pediatric EEG During Intractable Seizures 
Research Questions: 
1) How does modularity change before and during seizures?  
2) Do specific electrode groups consistently form communities during seizure periods? 
Description : This project analyzes how functional brain network organization changes before and during epileptic seizures using the CHB-MIT EEG dataset. EEG recordings are segmented into 5-second windows, and functional connectivity networks are constructed using pairwise correlations between electrodes. Graph-theoretical measures such as degree centrality, eigenvector centrality, density, and modularity are computed to characterize network structure. Community detection (Louvain/Leiden) is applied to identify modular organization across preictal and ictal states. Key metrics are partially implemented from scratch and benchmarked against standard Python libraries. The project aims to reveal structural reorganization patterns in brain networks during seizure events.
